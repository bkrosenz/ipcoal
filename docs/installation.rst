



Installation 
=============

You can install `ipcoal` using conda with the following command: 

.. code:: python 
	
	conda install ipcoal -c eaton-lab -c conda-forge

You can install additional [optional] analyses tools: 

.. code:: python
    
    conda install raxml -c bioconda


Dependencies
-------------

- numpy
- pandas
- numba
- msprime
- toytree
- seq-gen
